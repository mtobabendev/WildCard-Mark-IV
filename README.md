How to deploy Goofus-

## [Vercel](https://vercel.com)

Vercel is probably the most straight-forward way to deploy an app. All you need to do is import the Git repository to Vercel, and every time the repo updates, it will also update on the website automatically. Here's a complete [guide](https://vercel.com/docs/concepts/deployments/overview).

The only problem with Vercel is that it specializes in frontend deployment. If you have a backend, you would need to use serverless code. If you want to deploy to Vercel with Docker... well, you can't. No worries Sweetie. This was designed specifically without server dependencies for EXACTLY this reason.
